## ðŸ“‹ Step 1: Install Command Line Tools (REQUIRED)

**What is this?** Command Line Tools are needed for Python to work on Mac. It's NOT the full Xcode (24GB) - just the tools (~860MB).

### Option A: GUI Installation (EASIEST - Recommended)

1. A dialog box should appear on your screen asking to install Command Line Tools
2. Click **"Install"** (NOT "Get Xcode")
3. Enter your Mac password when asked
4. Wait 5-10 minutes for installation
5. You're done! âœ…

### Option B: Terminal Installation

If the dialog didn't appear, open Terminal and run:
```bash
xcode-select --install
```

Then follow the dialog that appears.

### Option C: Direct Install (If you prefer)

Open Terminal and run:
```bash
sudo softwareupdate --install "Command Line Tools for Xcode-16.4"
```
Enter your password when asked.
